  DISCUSSION: Cost Models for Multiple Treatments

  In Exercise 2, J=6 represents treatment arms across countries.

  Each arm requires separate implementation in a different country,

  implying costs that scale with the number of arms.

  Cobb-Douglas (beta=0.13, iota=0.075):

    - Costs scale smoothly with J (treatment arms)

    - Natural fit: each country arm is expensive to implement

    - For J=6: alpha_opt =0.010519

  Linear/FDA (cfshare=0.46, jbar=3):
