  Paper Table 3 q-values (endline 1, pooled):

================================================== 
  SUMMARY: Version A vs Version B
================================================== 

  Version A (10 outcomes, India):
    Cost structure: adding outcomes is CHEAP (cf_share ~ 0.82)
    => Strict correction needed (alpha* ~ 0.008, near Bonferroni)
    => Confirms 5 core results; physical health is borderline

  Version B (6 countries, pooled):
    Cost structure: adding a country is EXPENSIVE (cf_share ~ 0.10)
    => Minimal correction needed (alpha* ~ 0.040, near unadjusted)
    => Retains effects that Bonferroni would discard

  Same data, two framings, opposite corrections.
  This is the core insight of VWN (2026).
